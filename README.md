# Clinic Flow

This repository contains the source code for Clinic Flow, a tool to help chiropractors automate their clinics and save time.

## Getting Started

To get started, you'll need to have Node.js and npm installed. Then, you can clone this repository and run the following commands:

```bash
npm install
npm run dev
```

This will start the development server on http://localhost:3000.

## Features

*   **Toolkit of Toolkits:** A collection of pre-built automation recipes for common clinic tasks.
*   **Slash Commands:** A simple, powerful way to generate marketing copy, email sequences, and more.
*   **Admin Dashboard:** A central place to manage your API keys, view analytics, and more.
