# Outreach Tracking Methods

## 1. Simple Tracking Method
- Use a unique promo code for each partner.
- Partners distribute their unique promo code to their audience.
- Track redemptions or usage of each promo code to measure partner performance.

## 2. Advanced Tracking
- Generate a unique QR code or landing page URL for each partner.
- QR codes or URLs direct users to a customized landing page.
- Track visits, conversions, and other engagement metrics per partner.

## 3. Reconciliation at End of Month
- Generate reports summarizing promo code redemptions and QR code/URL visits.
- Compare reported data with internal sales or conversion data.
- Use the reports to calculate commissions, bonuses, or performance metrics for partners.
- Identify any discrepancies and investigate as needed.
